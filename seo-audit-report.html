No visible author byline on blog posts
Author exists in BlogPosting schema but is never rendered on-page. Google's Quality Rater Guidelines specifically look for visible author attribution. Add author name + link to /about/ in the blog post template.

2
Pagination pages indexed with duplicate meta descriptions
/blog/page/2/ through /blog/page/9/ all share the same meta description and are indexable. Add <meta name="robots" content="noindex, follow"> to prevent duplicate content in search index.
src/pages/blog/page/[num].astro
3
Blog cover aspect ratio mismatch
Cover images use width="1080" height="1080" (1:1 square) but are displayed with aspect-[2/1] CSS. This causes CLS and wastes browser layout computation. Update HTML attributes to match displayed dimensions.
src/pages/blog/[...slug].astro
4
No srcset on blog images
1080px images served to all viewport sizes. Mobile devices download 3x more pixels than needed. Add srcset with 640w/768w/1080w breakpoints or use Astro's <Image> component.
5
Privacy & Terms pages use single JSON-LD object
Both pages use a single schema object instead of an array and are missing BreadcrumbList. All other pages follow the array convention.
src/pages/privacy.astro, src/pages/terms.astro
